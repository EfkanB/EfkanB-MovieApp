# 🎬 MovieApp Landing Page

Bu proje, **HTML ve CSS** kullanılarak geliştirilmiş basit bir "MovieApp" tanıtım sayfasıdır.  
Amaç; kullanıcıya filmlerin ve dizilerin tanıtımını yapmak, kayıt olma ve plan seçme gibi adımları görselleştirmektir.

## 🚀 Özellikler
- Modern ve responsive tasarım
- Font Awesome ikonları
- Google Fonts entegrasyonu
- Landing page yapısı (Header, Showcase, How It Works, Features bölümleri)

## 📂 Kullanılan Teknolojiler
- HTML5
- CSS3
- Font Awesome
- Google Fonts

🎬 MovieApp Landing Page

This project is a simple "MovieApp" landing page developed using HTML and CSS.
